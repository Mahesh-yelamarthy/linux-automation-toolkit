# 🔧 Implemented Scripts

## Disk Usage Monitor

File:

```bash
monitoring/disk-monitor.sh
```

Features:
- monitors root disk usage
- checks disk threshold limits
- displays warning messages

## System Health Check

File:

```bash
health-checks/system-health.sh
```

Features:
- hostname information
- uptime monitoring
- memory usage statistics
- disk usage details
- running process visibility

---

## Log Cleanup Automation

File:

```bash
cleanup/log-cleanup.sh
```

Features:
- removes old log files
- automated cleanup workflow
- configurable retention period

---

## Home Directory Backup Automation

File:

```bash
backups/backup-home-directory.sh
```

Features:
- compressed backup generation
- timestamp-based backups
- automatic backup directory creation
